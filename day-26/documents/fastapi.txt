FastAPI is a modern, high performance web framework for building APIs with Python. It is built on top of Starlette for the web parts and Pydantic for data validation. FastAPI automatically generates interactive API documentation using Swagger UI and ReDoc. It supports asynchronous request handling, dependency injection and type hints, which allow for automatic request validation and editor autocompletion. FastAPI is commonly used to build REST APIs that need to be fast, well documented and easy to maintain.
