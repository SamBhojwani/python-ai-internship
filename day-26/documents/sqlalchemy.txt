SQLAlchemy is a Python SQL toolkit and object relational mapper that gives developers the full power of SQL while allowing them to work with Python objects instead of raw queries. It provides an ORM layer that maps Python classes to database tables, allowing developers to create, read, update and delete records using Python code. SQLAlchemy supports multiple database backends including SQLite, PostgreSQL and MySQL. It also provides a lower level Core API for developers who want more direct control over SQL execution.
