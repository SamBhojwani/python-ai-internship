Docker is a platform for developing, shipping and running applications inside lightweight, portable containers. A container packages an application along with all its dependencies so it behaves the same way regardless of where it runs. Docker images are built from a Dockerfile, which defines the steps needed to set up the application environment. Docker Compose allows multiple containers, such as an application and its database, to be defined and run together using a single configuration file. Docker is widely used to standardize development environments and simplify deployment.
