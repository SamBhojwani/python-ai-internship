FastAPI is a modern Python web framework used to build REST APIs quickly with automatic interactive documentation. To get started, install FastAPI and an ASGI server such as uvicorn, then define path operations using decorators like @app.get and @app.post. Pydantic models are used to define request and response schemas with automatic validation. Dependency injection using Depends allows shared logic such as authentication or database sessions to be reused across routes. FastAPI automatically generates Swagger UI documentation at the /docs endpoint.
